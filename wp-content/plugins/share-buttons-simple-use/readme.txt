A simple use plugin which displays the share buttons (Facebook Like button and Tweet button) on the bottom or on the top of the posts and pages.

== Description ==

Just activate this plugin and it will display the share buttons (Facebook Like button and Tweet button) on the bottom or on the top of the posts and pages.

In the admin panel you can configure the settings of the plugin. You can change the share buttons settings and set the display settings.

[Plugin Homepage](http://rubensargsyan.com/wordpress-plugin-share-buttons-simple-use)

== Installation ==

1. Upload the share-buttons-simple-use directory (including all files within) to the /wp-content/plugins/ directory
2. Activate the plugin through the Plugins menu in WordPress
